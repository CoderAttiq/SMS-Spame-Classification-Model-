# SMS Spam Detection using NLP

## Overview
This project builds an end-to-end NLP pipeline to classify SMS messages as spam or ham, designed for telecom spam filtering.

## Dataset
SMS Spam Collection v.1 (5,574 messages)

## Methods
- Text preprocessing (tokenization, stopwords, lemmatization)
- Feature extraction: BoW, TF-IDF, Word2Vec
- Models: Na√Øve Bayes, Logistic Regression

## Results
TF-IDF + Logistic Regression achieved the best overall performance.
